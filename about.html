About This Project

This project showcases a professionally designed frontend made with HTML, CSS, and JavaScript, backed by a powerful Flask (Python) backend. It demonstrates responsive design, clean code structure, and dynamic rendering using Jinja templates.

The goal of this UI/UX demo is to offer inspiration for developers building modern web applications, dashboards, portfolios, or AI-based tools using simple technologies.
